# JS Drag and Drop 🖱️

A drag-and-drop demo using HTML5 drag events and JavaScript. Good for learning DOM events and interactions.

## Features
- Drag and drop interactions
- Simple demo for learning

## Technologies Used
- HTML
- CSS
- JavaScript (Drag API)

## Project Structure
```
index.html — demo
styles.css — styles
script.js — drag/drop logic
```

## Installation / How to run
1. Clone repo.
2. Open `index.html` in a browser.
